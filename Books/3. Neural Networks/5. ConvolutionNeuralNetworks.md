CNNs
- Convolution Layers: São layers aonde filtros, também chamados de kernels, são aplicados as features de entrada, gerando uma nova saída que são nossas features' transformadas.
Cada kernel é uma matriz de N size, e possui valores diferentes em cada posição e cada kernel diferente, ou seja, são inicializados igual os pesos com valores aleatórios, o backpropagation arruma esses pesos
Podemos ter X Kernels, em cada layer convulucional, cada kernel vai transformar as features em features', ou seja se tivermos 10 kernels, teremos 10 FEATURES', que antes era apenas 1 feature

Imagine que temos 1 foto de um sapo, se tivermos 10 kernels, ele vai aplicar cada um, e assim gerar 10 fotos diferentes do sapo.
Se temos por exemplo (50x50x3) RGB, e o kernel 3x3, ele vai: 
	1. pegar o 3x3 do R, fazer a soma virando A, 
	2. depois do G, virando B, 
	3. depois do B, virando C 
	4. Depois ele faz A + B + C = D, e esse vai ser a feature map daquele pixel
https://towardsdatascience.com/using-convolutional-neural-network-for-image-classification-5997bfd0ede4/
- Pooling Layers: São layers aonde, usaremos outros kernels em nossas features', virando features'', ou seja, outra transformação, agora pegaremos o maior/média dos pixels ao redor daquele, assim diminuindo o seu tamanho e preservando estruturas ideias
	Se temos um (8x8), e um pooling de 2x2, isso vai pegar os maiores pixels em um quadrado 4x4

- Stride: Tamanho do passo que cada kernel da para direita/esquera ou baixo/cima
- Padding: São bordas que adicionamos as imagens para preservar seu tamanho orignal
- Flatten: Como normalmente usamos em matrizes, após todas as transformações vai ser uma matriz ainda, o flatten é o processo de transformar em um vetor para ir para uma MLP
